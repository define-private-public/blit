====== Tools ======

The Tools submodule of the [[widgets:drawing|Drawing Widgets]] are a set of well, tools, that are used to draw pixels onto [[:animation:cel|Cel objects]].  All of the Tools implement the [[widgets:drawing:tools:tool|Tool interface]].
